
## 采用MVVM架构

```plaintext
┌─────────────────────────────────────────────────────────┐
│  视图层（Views）                                        │
│  ├─ 主界面：TabBarView（标签页容器）                    │
│  ├─ 工作列表：PLJobsView（展示工作列表）                 │
│  ├─ 工作详情：PLJobView（查看/操作单个工作）             │
│  ├─ 发布/编辑工作：PostPLJobView、PostJobView           │
│  └─ 辅助视图：HUD（提示弹窗）、各类样式扩展（View+Style） │
├─────────────────────────────────────────────────────────┤
│  视图模型层（ViewModels）                               │
│  ├─ PLJobViewModel（管理工作数据、业务逻辑）             │
│  └─ TabBarViewModel（管理标签页切换状态）                │
├─────────────────────────────────────────────────────────┤
│  模型层（Models）                                       │
│  └─ PLJob（工作数据模型，包含标题、时薪、时间等属性）    │
├─────────────────────────────────────────────────────────┤
│  工具层（Utils）                                        │
│  ├─ 扩展：String+判断空值、Date+格式化、View+样式       │
│  ├─ 常量：布局间距（kStackSpacing）、样式配置            │
│  └─ 全局配置：AppDelegate（设置UISegmentedControl样式）  │
└─────────────────────────────────────────────────────────┘
